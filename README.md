This repository contains five mini-projects developed using Python: Chatbot, Tic-Tac-Toe Game, Image Captioning System, Recommendation System, and Face Detection System. These projects demonstrate core concepts of Artificial Intelligence, Machine Learning, Deep Learning, Computer Vision, and Python Programming. The applications include conversational interaction, game development, automated image understanding, personalized recommendations, and facial detection capabilities. The projects can be executed using Jupyter Notebook, Google Colab, or any Python-supported environment. They are designed for educational purposes, skill development, and practical implementation of modern AI techniques. Future enhancements may include advanced machine learning models, graphical user interfaces, real-time processing, and cloud deployment.
