# Face Detection System

## Overview
This project implements a Face Detection System using Python and OpenCV. It detects human faces in images or video streams and highlights them using rectangular bounding boxes.

## Features
- Face detection in images
- Multiple face detection support
- Real-time processing capability
- Easy-to-use implementation
- OpenCV Haar Cascade based detection

## Technologies Used
- Python
- OpenCV
- NumPy
- Jupyter Notebook

## How to Run
1. Install required libraries.
2. Open the notebook file.
3. Run all cells.
4. Upload or provide an image/video.
5. View the detected faces with bounding boxes.
