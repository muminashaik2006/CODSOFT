# Telugu Book Recommendation System

## Overview

The Telugu Book Recommendation System is a Python-based console application that recommends Telugu books based on the user's preferred decade and genre.

The system contains a curated collection of books categorized by different decades such as the 1980s, 1990s, 2000s, and 2020s. Users can select a decade and a genre to receive book recommendations along with their ratings.

---

## Features

* Decade-based book recommendations
* Genre-based filtering
* Displays book ratings
* Interactive command-line interface
* Input validation for decade and genre selection
* Multiple recommendation sessions

---

## How It Works

1. The system displays all available decades.
2. The user selects a preferred decade.
3. Available genres for that decade are displayed.
4. The user selects a genre.
5. The system recommends books from the selected category.
6. Ratings for each book are displayed.
7. The user can continue exploring or exit the application.
